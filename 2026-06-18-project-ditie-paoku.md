# 倒反天罡版地铁跑酷《没有目的地》

记录日期:2026-06-18 · 类型:项目档案

## 是什么
不是「人在地铁里跑酷」,而是「**地铁在跑酷**」——你就是地铁,在三条轨道上转弯穿梭,碾过被困在轨道上的人,没有目的地。把**电车难题倒过来玩**,并对原版《地铁跑酷》做**交互层 1:1 倒反复刻**(手势 ←→↑↓+双击全保留,语义倒反:金币→人、磁铁→人群磁铁、追你的督察→追不上的监察者)。

## 已锁定
- 调性 **B+C 冷面荒诞**(The Wire / No Country for Old Men,"世界只是运作,它不会等你");钝铁块无脸地铁、水泥冷色。
- **人即金币**(碾过=收集 + 强撞击反馈:hit-stop / 镜头震 / 布娃娃抛飞 / 慢动作 + 尸堆)。
- **碾人永不致败**,只有撞「会让机器停下的硬障碍」脱轨才败。
- 招牌机制**「它不会等你」**(岔路不暂停,不操作=保持当前道)。
- 收场:无限 + 脱轨失败,"这趟到此为止。下一趟照常发车。"
- 血腥度:风格化红块默认 + 设置可切写实(都做)。
