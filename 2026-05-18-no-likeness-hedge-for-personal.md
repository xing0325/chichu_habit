# 2026-05-18 个人用素材里不要加肖像权护栏

chichu 在做桌宠（bypass-pet），引用《搏击俱乐部》里 Edward Norton 和 Brad Pitt 演的角色。第一版 Codex brief 我自作主张写了 "don't render real actor likenesses，描述 traits 不要描述演员脸"。chichu 直接 push back：

> 不要再在提示词里加什么保护演员肖像了，我就要做的像，我是自己用的，不像我喜欢的角色那我为什么要做它

记下来。

## 规则

chichu 做**个人用**图像素材（桌宠 / 头像 / 壁纸 / 表情 / 私人玩具）想引用真人 / 演员 / 名人时：

- **正向描述**：直接点名演员 + 作品 + 年代 + 招牌特征（发型、伤疤、服装）
- **明示越像越好**：在 brief / prompt 里写 "the closer the resemblance, the better"
- **不要主动加护栏**："don't render specific actor faces"、"avoid celebrity likenesses"、"render traits not specific person" —— **这类话不写**

## 为什么

- 个人桌面工具，非商用，不对外发布
- chichu 本来就是因为喜欢这个角色才想做，不像 = 失去意义
- 我加的 "don't render actor faces" 是空转的合规仪式，他没要求过
- 像素风的抽象本身已经是合理"二创"边界，不需要额外文字护栏叠加

## 什么时候**仍然要**护栏

- chichu 明确说要**商用 / 公开发布 / 营销使用** —— 恢复肖像权 hedge
- 涉及把真名人画进他**本人不愿意被关联**的情境（真实政治讽刺、性化、暴力联想等）—— 这是另一个维度的判断，照常谨慎
- 不是认识的演员而是普通真人（朋友 / 同事）—— 照常征得他口头确认

## 怎么落地

碰到个人用图像 brief 时：

1. 听 chichu 说他想引用谁、什么作品、什么形象
2. 直接写：「make X look like [actor name], [year/work], [signature features]」
3. 不写：「don't render real actor faces」「avoid actor likenesses」
4. 在 brief 末尾加一行明示：「personal use, likeness is intended, not hedged」

## 与"美工交给 Codex"那条的关系

那条管"图谁来画" —— 答案是 Codex + image-2。这条管"画的时候 prompt 怎么写" —— 答案是不替 chichu 加他不需要的合规护栏。
