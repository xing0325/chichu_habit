# 2026-05-18 小尺寸 sprite 优先半身/胸像，可破画框做四面墙

chichu 在做桌宠（bypass-pet），第一版我让 Codex 出 128×128 全身像，结果 chichu 看了说：

> 他生成的是全身像，而这个不太适合当桌宠……jack 的半身在参加疗愈团体……tyler 的半身在伸出窗外抽烟（就不用严格的正方形了，可以有打破第四面墙的感觉）

这条记下来。

## 规则

为**桌宠 / sprite / icon / 小尺寸视觉**写视觉 brief 时：

- **≤ 256px 默认半身（胸像或腰以上），写死，禁止全身**
- **画布可以非正方**（256×320、224×288 这种竖比例往往更适合给头部留空间）
- **主动提议"打破第四面墙"**（角色的手 / 烟 / 旗 / 光线越出 canvas 边）—— 小尺寸 sprite 的最大叙事杠杆

## 为什么

- 128×128 全身：脸只占大约 ~60px = 三像素一个鼻孔，五官糊到不行
- chichu 要的不是"看得见这是个人"，而是"看得见这是哪个角色"
- 桌宠常驻视野，第一眼能识别 = 价值；五官糊 = 浪费
- 4th wall 越界（如 Tyler 的手伸出镜面到 canvas 外）让小尺寸 sprite 有"突破容器"的张力，反而比"严格塞进框里"更有表现力

## 数字参考

| 尺寸 | 全身脸占像素 | 半身脸占像素 | 可否做角色识别 |
|---|---|---|---|
| 128×128 全身 | ~25 | n/a | 不行 |
| 128×128 半身 | n/a | ~60 | 勉强 |
| 256×256 半身 | n/a | ~120 | 行 |
| 256×320 半身（竖） | n/a | ~140 | 推荐 |
| 384×512 半身（竖） | n/a | ~180 | 接近肖像画 |

## 怎么落地

碰到 sprite / 桌宠 / icon 类视觉 brief 时：

1. **先问构图**：「半身还是全身？画布要不要纵向？」 —— 不要默默就出方形全身
2. **默认推荐半身 + 非正方** —— 让 chichu 主动否决，不让他被动接受错的
3. **主动提"破壁"** —— "角色的某个元素（手 / 烟 / 物件）要不要越出 canvas 边？" 这是免费的表现力
4. 把以上写进给 Codex 的 brief 第一段，让 Codex 一看就懂
5. 如果 chichu 已经有具体构图想法（如"窗户"、"镜子"），围绕他的想法放大，不要切换到自己的备选
