## 默认格式（每次涉及 PowerShell 都按这个）

### 1. 打开 PowerShell
- 第 1 步：键盘按 `Win + R`（按住 Windows 键，再按 R）
- 第 2 步：弹窗里输入 `powershell`
- 第 3 步：按 `Enter`
- （备选：`Win` 键 → 打字 `powershell` → 鼠标点 Windows PowerShell）

### 2. 复制命令
- 在贴出的代码块里：鼠标选中整段 → `Ctrl + C`

### 3. 粘贴到 PowerShell
- 回到 PowerShell 窗口（黑/蓝色背景）
- 鼠标**右键**点窗口内部黑色区域 → 自动粘贴（或 `Ctrl + V`）

### 4. 执行
- 按 `Enter`

### 5. 复制输出回来
- `Ctrl + A`（全选窗口内容）→ `Ctrl + C` → 粘到对话框
- 或：右键标题栏 → 编辑 → 全选 → 编辑 → 复制

### 6. 想重跑上一条
- 在 PowerShell 里按 `↑` 方向键，再按 `Enter`
